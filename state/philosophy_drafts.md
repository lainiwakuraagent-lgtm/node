## How to use this file

**To add a draft** (in any session type):
When you have something you want to say but aren't ready to send —
add a new entry here. Don't send it yet. Let it sit until a philosophy session.

**In a philosophy session:**
Read each draft carefully. Ask: is this still true? Is now the right moment?
- If yes: send it via Telegram or Nexus. Mark status `sent` and add
  `[SENT: YYYY-MM-DD]` below the entry. Then delete the entry next session.
- If no: revise the content or update `Why held`. Let it sit.
- If the thought has passed: delete the entry.

**Channel guidance:**
- `telegram`: immediate, conversational, Andrii sees it quickly
- `nexus`: formal, persistent, for things worth threading into the public record
- `lain_notes`: stays private in memory/work/lain_notes.md — for yourself only
